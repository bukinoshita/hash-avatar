# hash-avatar

> Hash avatar algorithm

## Install
```
$ npm install --save hash-avatar
```

## Usage
```js
import hashAvatar from 'hash-avatar'

hashAvatar('uid', { size: 100 })
// => It will return a SVG
```

## API
### hashAvatar(uid, { size })

#### uid
Type: `string`<br/>
Required

#### size
Type: `number`<br/>
Required
